Fonts indic archaic is an collection of ancient 
indic scripts for Slackware Linux.

The indic scripts supported by this collection are
1. Ahom
2. Bhaisukui
3. Brahmi
4. Dives-Akuru
5. Grantha
6. Kaithi
7. Kharoshthi
8. Khojki
9. Khudawadi
10. Mahajani
11. Modi
12. Multani
13. Nandi Nagari
14. Sharada
15. Siddham
16. Sylhot Nagarai
17. Trihuta
